# rostovTvRadioStationsWebsite

Пробный проект на React + Django

Планируется реализовать:
- Интерактивный список частот ТВ и радио
- Добавить онлайн-потоки
- Частоты по городам/районам
- Список лицензий

План максимум:
- Создать административную часть для возможности редактирования таблиц
- Возможность дать пользователю роль редактора определенного города
- Хранилище записей
- История по частотам

Возможно география будет расширена
