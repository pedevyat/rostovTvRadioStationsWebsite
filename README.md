# rostovTvRadioStationsWebsite

Пробный проект на React + Django

Планируется реализовать:
- Интерактивный список частот радио и их технические и географические характеристики
- Добавление онлайн-потоков
- Частоты по городам/районам


План максимум:
- Возможность дать пользователю роль редактора своего города
- Хранилище видео- и аудиозаписей

Возможно география будет расширена
